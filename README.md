# DexTraderPro Alpha

Public alpha operations repository for DexTraderPro.

This repo is used for:
- binary release distribution via GitHub Releases
- user issue tracking (bugs, feature requests, alpha feedback)
- release notes and known issues during alpha

- https://github.com/madiver/dextraderpro-alpha/releases/latest

## Platform Notes

Current alpha binaries are published per release as:
- macOS installer/package
- Windows installer/package

Each release includes:
- build/version details
- highlights
- known issues
- rollback/workaround notes when needed

## Crash and Diagnostic Paths

When reporting a crash, include:
- app version and platform
- reproduction steps
- latest log/crash file if available

Common paths:
- macOS crash reports: `~/Library/Logs/DiagnosticReports/`
- Windows crash dumps: `%LOCALAPPDATA%\\CrashDumps\\`
