### Channel
- Private alpha (invite waves)

### Release status
- macOS installer is ready in this release.

### Highlights
- Introduced the first DexTraderPro alpha with a trader-first desktop workflow for on-chain execution.
- Added direct wallet-to-venue submission without a DexTraderPro relay path.
- Added initial scanner coverage in alpha.
- Added reliability indicators for stale and degraded data states.
