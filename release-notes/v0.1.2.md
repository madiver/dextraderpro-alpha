## DexTraderPro Alpha Release v0.1.2

### Channel
- Private alpha (invite waves)

### Release status
- macOS installer is ready in this release.
- Windows installer for `v0.1.2` is pending and will be added in a follow-up update.

### Highlights
- Improved starter-layout restore behavior with monitor-agnostic reseed semantics and deterministic `Reset Layout` behavior that applies to the active layout slot.
- Reduced UI hitching in Time & Sales and Price Structure by moving heavy frame/rebuild preprocessing off the UI thread and applying incremental updates where possible.
- Hardened execution/datafeed determinism for multi-network sessions (cycle isolation, stale-callback rejection, and deterministic submission/retry terminalization).
- Hardened signing-credential lifecycle by enforcing local-auth protection for signing-credential removal (fail closed when local auth is unavailable).

### Upgrade notes
- `Reset Layout` now reseeds the current active layout slot to the starter baseline instead of forcing a switch to a specific saved layout name.
- Existing saved user layouts are preserved; baseline reseed applies deterministic dock arrangement with monitor-agnostic placement rules.

### Known issues
- Opening a new floating window can still briefly blank/freeze the main UI in some environments while the float operation (`setFloating(True)`) completes.

### Workarounds
- Avoid opening many new floating windows in rapid succession on lower-performance machines.
