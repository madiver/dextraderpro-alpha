## DexTraderPro Alpha Release v0.1.2

### Channel
- Private alpha (invite waves)

### Release status
- macOS installer is ready in this release.
- Windows installer is ready in this release.

### Highlights
- Improved status and health indicators so connecting, unknown, and inactive states are clearer across the app.
- Improved startup recovery so session interruptions and in-flight Order Entry work recover more cleanly after restart.
- Reduced UI hitching in Time & Sales and Price Structure during heavier live-update activity.
- Improved layout reset behavior so the current layout refreshes more predictably without disturbing saved layouts.
- Improved multi-network stability during feed recovery and submission retry scenarios.
- Improved wallet credential protection by requiring local authentication before removing signing credentials.

### Upgrade notes
- `Reset Layout` now refreshes the active layout instead of switching you to a different saved layout.
- Existing saved layouts are preserved.
- In-flight Order Entry work may now resume and reconcile more reliably after restart.

### Known issues
- Opening a new floating window can still briefly blank/freeze the main UI in some environments while the float operation (`setFloating(True)`) completes.

### Workarounds
- Avoid opening many new floating windows in rapid succession on lower-performance machines.
