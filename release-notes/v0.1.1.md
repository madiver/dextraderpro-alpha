## DexTraderPro Alpha Release v0.1.1

### Channel
- Private alpha (invite waves)

### Release status
- macOS installer is ready in this release.
- Windows installer is ready in this release.

### Highlights
- Improved wallet secure-storage behavior on macOS so signing credentials are more reliable.
- Added session authentication preference controls in Application Settings.
- Added an explicit logout flow in desktop menus.
- Improved open-window responsiveness, especially when opening floating windows for the first time.

### Included installers
- macOS: `DexTraderPro-0.1.1.dmg` (attached release asset)
- Windows: `DexTraderPro-windows-x64-v0.1.1-setup.exe` (attached release asset)

### Upgrade notes
- Existing sessions created before this release may keep their original expiry timing until renewed.
- If you change the session-auth preference in Application Settings, the new preference applies immediately.

### Known issues
- Opening a new floating window can still briefly blank/freeze the main UI in some environments (more noticeable on slower Windows VMs).

### Workarounds
- Avoid opening many new floating windows in rapid succession on lower-performance machines.
