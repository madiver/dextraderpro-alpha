## DexTraderPro Alpha Release v0.1.1

### Channel
- Private alpha (invite waves)

### Release status
- macOS installer is ready in this release.
- Windows installer is ready in this release.

### Highlights
- Hardened wallet secure-storage behavior on macOS with improved signing credential availability and packaging/signing alignment.
- Added session authentication preference controls in Application Settings (`Every login`, `5 minutes`, `15 minutes`, `1 hour`, `8 hours`, `12 hours`, `24 hours`) with immediate runtime apply.
- Added explicit logout flow in desktop menus to clear local session state and exit.
- Improved open-window responsiveness by reducing first-open floating dock overhead and adding timing instrumentation for open-path diagnostics.

### Upgrade notes
- Existing sessions created before backend TTL updates keep their original expiry. New sessions use the current backend max TTL policy.
- If you select a different session-auth preference in Application Settings, the new preference applies immediately.

### Known issues
- Opening a new floating window can still briefly blank/freeze the main UI in some environments (more noticeable on slower Windows VMs).

### Workarounds
- Avoid opening many new floating windows in rapid succession on lower-performance machines.
