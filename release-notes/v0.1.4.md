## DexTraderPro Alpha Release v0.1.4

### Channel
- Private alpha (invite waves)

### Release status
- macOS installer is ready in this release.
- Windows installer is ready in this release.

### Highlights
- Added the new `AI Trading Partner` panel for decision-support workflows inside the desktop app.
- Improved linked-window trading workflows with color-linked pool synchronization across Time & Sales, Price Structure, Chart, and Order Entry.
- Improved Positions for mixed-quote portfolios with clearer grouping, quote context, and subtotal handling.
- Improved licensing and startup recovery so activation, retry, and failure states are clearer and less disruptive.
- Improved layout reset behavior so `Reset Current Layout...` refreshes the active layout more predictably while preserving saved layouts.
- Improved Order Entry recovery and submission tracking so restart/resume behavior is more reliable after interrupted flows.
- Reduced UI hitching in Time & Sales and Price Structure during heavier live-update activity.
- Improved wallet credential protection by requiring local authentication before removing signing credentials.

### Upgrade notes
- `Reset Current Layout...` now refreshes the active layout instead of switching you to a separate saved layout.
- Window titles now include the build identifier after the version, which helps with support and issue reporting.
- Time & Sales now starts in `Connecting` until the feed is actually live.
- If you use mixed-quote Positions views, review the new grouping controls and subtotal rows after upgrade.
- If you use AI Trading Partner, review the panel states and apply-to-ticket behavior after upgrade.

### Known issues
- Opening a new floating window can still briefly blank/freeze the main UI in some environments while the float operation (`setFloating(True)`) completes.

### Workarounds
- Avoid opening many new floating windows in rapid succession on lower-performance machines.
